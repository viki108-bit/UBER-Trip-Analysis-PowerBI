1️⃣ Business Purpose

To analyze Uber trip performance across multiple cities and time periods, helping identify patterns in customer behavior, trip demand, vehicle utilization, revenue growth, and operational efficiency. This dashboard supports data-driven decision-making for improving customer experience and optimizing fleet management.

2️⃣ Data Source

The analysis is based on a structured dataset containing Uber trip records, including fields such as Trip ID, Pickup Date, Vehicle Type, Payment Type, Passenger Count, Trip Distance, Pickup Location, and Booking Value.

3️⃣ Time Period Covered

The dataset covers the period 1st June 2024 to 30th June 2024.

4️⃣ Main KPIs Used

Total Bookings

Total Booking Value

Average Booking Value

Total Trip Distance

Average Trip Distance

Average Trip Duration

Farthest Trip

Most Frequent Pickup and Drop-off Points

5️⃣ Filters / Slicers

Date Range

City

Vehicle Type

Payment Type

6️⃣ Update / Refresh Frequency

The dashboard can be refreshed monthly, depending on availability of updated trip records.

This dashboard is built for educational and analytical purposes only. It does not represent official Uber data, and any resemblance or similarity is purely coincidental. The dataset used is not sourced from Uber’s internal systems
